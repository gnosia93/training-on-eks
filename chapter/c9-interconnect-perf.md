### 인터케넥트 타입 ### 

* Node Interconnect: 서버와 서버 사이의 연결 (EFA, ENI, InfiniBand 등)
* GPU Interconnect: GPU와 GPU 사이의 연결 (NVLink)
* System Interconnect: CPU, GPU, 메모리 간의 연결 (PCIe)
